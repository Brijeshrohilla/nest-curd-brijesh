# NEST CURD

A simple CURD app using NodeJS and React (Redux Toolkit).
Which is capable for play around some CURD operations.

LOCAL URL: http://localhost:3000

## How to run App

```bash
# top directory
$ npm start
```
## Frontend

- React
- Axios
- chakra Ui
- Typescript
- Redux Toolkit

## Backend

- Nest
- Nodejs
- Typescript
- MongoDB & Mongoose
